# ❤️ Heart Attack Prediction System

A standalone desktop application developed in Python using Tkinter for GUI, Logistic Regression from scikit-learn for prediction, and MySQL for data storage.

## 🚀 Features

- 🎛️ Input patient details via a user-friendly GUI
- 📊 Visualize user data with multiple matplotlib plots
- 🧮 Predict heart disease risk using Logistic Regression
- 💾 Store records in a MySQL database
- 🗃️ Offers save, clear, info, and logout controls
